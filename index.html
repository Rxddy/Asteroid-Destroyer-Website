About Me

Hi! I'm Rusanth Pirabakaran, a passionate game developer with a love for retro gaming. I have always been fascinated by classic arcade games from the 80s and 90s, which is why I decided to create "Asteroid Destroyer," a retro-style space shooter game.

I used Python, HTML, CSS, and Java to create both the website and the game. I'm constantly learning new techniques to improve my skills.

In addition to my love for game development, I enjoy playing sports such as volleyball and video games like Valorant. I created this website to showcase my work and connect with other gamers and developers.

Game Description

Asteroid Destroyer is an exciting retro-style space shooter where players navigate through the galaxy, battling waves of enemy ships and dodging incoming asteroids.

With fast-paced action, classic pixel graphics, and immersive sound effects, this game brings the thrill of old-school arcade space battles to life.
